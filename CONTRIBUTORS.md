## How this file is maintained

There are two recommended ways to add contributors:

1. Use the All Contributors bot
   - In a PR or issue comment, request the bot to add a contributor, e.g.:
     @all-contributors please add @username for code, doc, infra
   - The bot will open a PR adding the contributor to this file.

2. Manually (maintainers)
   - Open a small PR that:
     - Adds the contributor entry under the "Contributors" section.
     - Keeps the list in alphabetical order by GitHub username (or by surname if usernames are not known).
     - Uses the format shown in the example below.
